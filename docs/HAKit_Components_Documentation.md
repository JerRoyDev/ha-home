# HAKit Components Documentation

This document provides a comprehensive list of components available in the `@hakit/components` library, with links to their official documentation and stories.

---

## Provider & Layout

- **[ThemeProvider](https://shannonhochkins.github.io/ha-component-kit/?path=/docs/components-themeprovider--docs)**: Essential provider for global styles and CSS variables.
- **[Group](https://shannonhochkins.github.io/ha-component-kit/?path=/docs/components-group--docs)**: Automatically layouts children in a row with a predefined gap, supports collapsing.
- **[Row](https://shannonhochkins.github.io/ha-component-kit/?path=/docs/components-shared-row--docs)**: Standard flex row layout.
- **[Column](https://shannonhochkins.github.io/ha-component-kit/?path=/docs/components-shared-column--docs)**: Standard flex column layout.
